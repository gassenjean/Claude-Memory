# Templates Claude-Memory

Esta pasta contém templates reutilizáveis para otimizar interações com Claude usando sistemas de memória persistente.

## 📋 Templates Disponíveis

### **🧠 Core Memory System**

### 1. **Protocolo de Memória Claude** (`protocolo-memoria-claude.md`)
- **Função:** Instruções base para ativação do sistema de memória
- **Uso:** Cole nas instruções do projeto Claude
- **Benefício:** Continuidade e personalização automática das conversas

### **🎯 TDAH & Produtividade**

### 2. **Aprendizado IA para TDAH** (`aprendizado-ia-tdah.md`)
- **Função:** Sistema estruturado para aprender IA generativa com TDAH
- **Uso:** Framework completo de estudos, experimentos e tracking
- **Benefício:** Transforma dispersão em foco estratégico para IA

### 3. **Configuração MCP TDAH** (`configuracao-mcp-tdah.md`)
- **Função:** Setup completo do MCP para casos específicos TDAH
- **Uso:** Guia passo-a-passo com workflows e troubleshooting
- **Benefício:** Sistema otimizado para hiperfoco e múltiplos projetos

### **🚀 Aplicações Práticas**

### 4. **IA nos Projetos Gassen** (`ia-projetos-gassen.md`)
- **Função:** Templates para aplicar IA em projetos específicos
- **Uso:** Névoa, Gabriele, Granja, Evangelismo, Agrofloresta, HQ
- **Benefício:** Roadmap integrado com sinergias cross-project

### **💡 Criatividade & Trabalho**

### 5. **Questionário Criatividade-Paralisia** (`questionario-criatividade-paralisia.md`)
- **Função:** Framework para superar bloqueios criativos
- **Uso:** Sessões de desbloqueio e brainstorming
- **Benefício:** Estrutura para navegar momentos de paralisia criativa

### 6. **Sessão de Trabalho** (`sessao-trabalho.md`)
- **Função:** Template para sessões produtivas focadas
- **Uso:** Início de sessões de trabalho estratégico
- **Benefício:** Maximiza produtividade com estrutura clara

## 🚀 Como Usar

### **Para Iniciantes em IA + TDAH**
1. **Configure primeiro:** `protocolo-memoria-claude.md`
2. **Setup específico:** `configuracao-mcp-tdah.md`
3. **Estruture aprendizado:** `aprendizado-ia-tdah.md`
4. **Aplique nos projetos:** `ia-projetos-gassen.md`

### **Para Usuários Avançados**
1. Combine templates para workflows personalizados
2. Adapte prompts para seu contexto específico
3. Use sinergias cross-project identificadas
4. Contribua com novos templates baseados em sua experiência

### **Para Casos Específicos**
- **Bloqueio criativo:** Use `questionario-criatividade-paralisia.md`
- **Sessão produtiva:** Use `sessao-trabalho.md`
- **Aprendendo IA:** Use `aprendizado-ia-tdah.md`

## 🔧 Configuração

### **Pré-requisitos**
- Claude 4 (Sonnet ou Opus)
- MCP Memory Server (recomendado)
- Node.js ou Docker para MCP

### **Setup Rápido (10 minutos)**
1. **Instale MCP:** Siga `configuracao-mcp-tdah.md`
2. **Configure Claude:** Cole `protocolo-memoria-claude.md` nas instruções
3. **Primeira sessão:** Use `aprendizado-ia-tdah.md` ou `ia-projetos-gassen.md`
4. **Inicie conversa:** Claude dirá "Lembrando..." automaticamente
